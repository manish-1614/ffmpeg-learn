# FFmpeg Learn

A tutorial and command explorer for learning FFmpeg programmatically.

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Preview Production Build

```bash
pnpm preview
```

## Project Structure

- `videos/` - All learning videos in one place
- `src/` - Application source code
- `public/` - Static assets
